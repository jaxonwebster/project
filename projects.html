CineNiche

My team and I placed 2nd in a coding competition to create a functional streaming service that recommends movies based on watch history and preferences. Under a strict deadline, we completed this project in 4 days and presented to professionals in the industry. We used Python, .NET, C#, Javascript, HTML, CSS, and a predictive model. Users can create an account, rate movies, and get suggesetions based on their history. Infinite scrolling also allows users to use the app for longer. Admin can edit the database of movies and manage users.
